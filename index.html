Algoduel

How to Play

In Algoduel, your fighter's actions are controlled by a simple algorithm.

  • attack(): Deals damage to the enemy.
  • heal(): Restores your fighter's health.
  • defend(): Reduces incoming damage next turn.

Variables you can use:

  • health: Your current health.
  • enemy: The enemy fighter object (e.g., enemy.health).

Example algorithm:

if (enemy.health < 30) {
    attack();
} else if (health < 50) {
    heal();
} else {
    defend();
}

Good luck, and may the smartest strategy win!
